# sitecore-jss-grid-components
## Description
A demo to show how to publish Sitecore JavaScript Services (JSS) components as NPM packages.

## Usage
To view or work with components
```
npm install
jss start
```

To generate npm package
```
npm run webpack
npm pack
```
